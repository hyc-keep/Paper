### 2.1 `Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`

用途定位：

- 一篇很新的直接腺体分割论文，适合补“attention + multi-scale context fusion”这条近期路线。
- 更适合做近期 related work 和“如果要再做结构增强，可以怎么讲”的补强证据，不一定适合直接作为主 baseline。

可直接提取的关键信息：

1. 论文摘要直接指出：作者提出 `MAC-Net`，将 `multi-scale feature fusion`、`attention-guided contextual decoding` 和 `U-Net backbone` 结合，用于应对腺体细粒度结构、染色差异和不同分化阶段的异质性。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.1

2. 数据设置不是只在单一数据集内验证，而是 `EBHI-Seg` 上训练，再在 `GlaS` 上做 cross-dataset 泛化验证；摘要中给出的结果是 `Dice 95.08% / IoU 90.92% / Precision 95.83% / Recall 95.65%`。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.1

3. 方法动机部分明确批评普通 `U-Net`：其 skip connection 对低层空间信息与高层语义信息融合不足，难以处理 varying gland sizes、intra-class variation、staining artifacts 与复杂边界。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.2

4. `MAC-Net` 的关键模块包括：encoder 侧与 skip 连接上的 `channel attention`、bottleneck 处的 `multi-scale context aggregation`，以及更密集的 lateral connection。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.3, p.4, p.5

5. 损失函数不是单一 Dice，而是 `Dice + Focal + Edge loss` 的 hybrid loss；作者明确说这是为了同时缓解 class imbalance 并强化 boundary delineation。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.3, p.4

6. 论文还显式加入 `Grad-CAM` 与 `Monte Carlo Dropout` 不确定性分析，说明这篇不只是追求分割分数，也试图补“可解释性”和“低置信样本提醒”。
   - 来源：`Enhanced_colorectal_gland_segmentation_through_multi-scale_attention_and_contextual_feature_fusion_2026.pdf`, p.3

对当前实验最有用的落地点：

- 它适合放在你后面 related work 的“近期 attention/context fusion 腺体分割路线”里。
- 如果你后续不做很重的 context decoder，这篇也能作为“别人这样做过，但我们不走这么重的融合结构”的对照说明。

### 2.2 `Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`

用途定位：

- 这篇非常重要，因为它把腺体分割和 `IHC biomarker quantification` 直接接起来了。
- 对你的写作价值很高，尤其适合支撑“腺体分割不仅是分割任务，还能服务后续定量病理分析”。

可直接提取的关键信息：

1. 摘要直接说明：论文目标不是普通 gland mask，而是自动分割 `glandular epithelium`，用于对 colorectal tissue 的 `IHC biomarker quantification` 做 compartmentalised analysis。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.1

2. 方法设计强调跨染色适用性：既面向 `H&E`，也面向 `IHC`；作者只使用经 color deconvolution 提取出的 `haematoxylin` 通道，以提高对不同 IHC marker 的泛化能力。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.1, p.4, p.5

3. 论文明确使用 `GlaS` 数据来和已有方法比较，同时又自建了 IHC 数据集，并指出模型在新 IHC 数据上只需少量额外样本 fine-tune 即可适配。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.1, p.2

4. 新网络结构融合了 `DCAN`、`U-Net` 和 `ResNet identity mapping` 的思路，同时保留 object segmentation 与 edge segmentation 双输出，但比更宽更深的复杂模型更节省资源。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.3, p.4, p.5

5. 实现细节里，作者使用 `480×480` 输入、`Adam(lr=0.0003)`、`batch size=2`，并重度依赖 realistic data augmentation 来增强对 stain 和 acquisition variation 的鲁棒性。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.5, p.6

6. 论文背景部分明确指出，IHC 定量如果能够限制在 glandular epithelium 等特定组织 compartment 内，会比整张切片粗放统计更有信息量。
   - 来源：`Segmentation_of_glandular_epithelium_in_colorectal_tumours_to_automatically_compartmentalise_IHC_biomarker_quantification_A_deep_learning_approach_2018.pdf`, p.2

对当前实验最有用的落地点：

- 这篇非常适合放进引言和讨论里，解释为什么腺体分割有下游价值。
- 也适合在 future work 里支撑“后续可面向 IHC / biomarker compartmentalisation 扩展”。

### 2.3 `A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`

用途定位：

- 传统方法里比较有代表性的一篇，突出“只利用 nuclei 空间排布，也能做 gland segmentation”。
- 很适合补“深度学习之前/之外，结构先验一直很重要”这条脉络。

可直接提取的关键信息：

1. 摘要开头就强调：intestinal gland morphology 对炎症性肠病严重程度和 CRC 恶性程度/预后评估都很重要，因此需要准确的 gland segmentation。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.1

2. 方法核心不是 texture 或颜色，而是基于 `mathematical morphology` 对 nuclei 的空间分布进行建模，并把 gland 按 appearance 分成 `hallow glands` 和 `solid glands` 两类。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.1, p.2

3. 作者先做 stain normalization 和 nuclei extraction，再构造 `density map / solidity map / girth map` 来分别刻画空心区域、上皮层连通性和围绕中心区域的包围关系。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.2, p.3, p.4, p.5

4. 这篇很有意思的一点是：它不是靠 active contour 或复杂 region growing 去找边界，而是尽量直接利用 epithelial layer 本身来界定 gland 结构。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.5

5. 评估使用两套 H&E 数据：`Warwick` 数据集 `17` 张图 `255` glands，和 `APHP-Sorbonne` 数据集 `64` 张图 `1824` glands；指标采用 `F1-score` 和 `Object-Dice`。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.5, p.6

6. 结果表中总成绩为 `F1 0.8328 / Object-Dice 0.8898`，作者也强调该方法的一个优势是可扩展到 `immunohistochemistry`、`fluorescence` 及其他 glandular tissues。
   - 来源：`A_Structure-Based_Approach_for_Colon_Gland_Segmentation_in_Digital_Pathology_2016.pdf`, p.6, p.7

对当前实验最有用的落地点：

- 这篇可以帮你在写作时说明：腺体分割从来不是单纯像素分类问题，结构排布和细胞组织关系一直都是关键。
- 它也能支撑你后面强调 morphology-aware 设计不是拍脑袋。

### 2.4 `Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`

用途定位：

- 更早期、但很经典的一篇图结构思路论文。
- 适合支撑“在深度学习之前，腺体任务已经非常强调 object-level 组织关系，而不仅是 pixel-level 分类”。

可直接提取的关键信息：

1. 摘要明确指出：腺体分割对于自动分析腺体组织活检很重要，但 staining、fixation、sectioning 带来的变异会让 gland appearance 差异很大。
   - 来源：`Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`, p.1

2. 方法核心是把图像分解成 primitive objects，并用 `object-graphs` 表征这些对象的组织关系；作者特别强调这是一种 object-based，而不是单靠 pixel-based 的路线。
   - 来源：`Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`, p.1, p.2

3. 整个算法是三步 region growing：先基于 lumen object 判定 gland seeds，再借助 nucleus graph 生长 gland 区域，最后再做 false gland elimination。
   - 来源：`Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`, p.2, p.3

4. 作者指出 object-based 结构相较 pixel-based 信息对 tissue artifacts 和 staining / fixation 变化更稳健，这是这篇方法学上的核心论断。
   - 来源：`Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`, p.2

5. 在 `36` 位病人的 colon tissue 上，region-growing 后平均 segmentation accuracy 为 `82.57%`，经 false gland elimination 后提高到 `87.59%`。
   - 来源：`Automatic_Segmentation_of_Colon_Glands_Using_Object-Graphs_2010.pdf`, p.3

对当前实验最有用的落地点：

- 这篇很适合补 related work 的历史脉络。
- 也能帮助你在讨论里强调：腺体任务天然适合 object/graph/structure 视角。

### 2.5 `Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`

用途定位：

- 一篇很值得保留的无监督腺体分割论文。
- 它对你最大的价值不是“要不要复现”，而是证明 morphology prior 即使在无监督场景里也非常关键。

可直接提取的关键信息：

1. 摘要直接指出：该文试图在 `no manual annotations` 条件下做 `unsupervised gland segmentation`，核心难点是现有 USS 方法会出现 over-segmentation 或 under-segmentation。
   - 来源：`Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`, p.1

2. 论文提出的关键先验是：每个 gland 通常由高灰度边界区域包围内部 epithelial tissues，因此可借助 morphology cue 去选择 gland sub-region proposals。
   - 来源：`Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`, p.2, p.3

3. 方法由 `Selective Proposal Mining (SPM)` 和 `Morphology-aware Semantic Grouping (MSG)` 两部分组成，不是直接从全图硬聚类 gland，而是先找子区域、再做语义聚合。
   - 来源：`Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`, p.3, p.4, p.5

4. 数据评估覆盖 `GlaS` 和 `CRAG`，其中 `CRAG` 结果来自在 `GlaS` 上训练后直接迁移，作者借此强调方法对更不规则 malignant glands 的适应性。
   - 来源：`Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`, p.6

5. 结果表显示：在 `GlaS` 上 `F1 78.26 / Dice 77.09 / mIoU 62.72`，在 `CRAG` 上 `F1 77.43 / Dice 77.26 / mIoU 65.89`，并且 `mIoU` 比第二好的无监督方法高 `10.56%`。
   - 来源：`Morphology-Inspired_Unsupervised_Gland_Segmentation_via_Selective_Semantic_Grouping_2023.pdf`, p.7

对当前实验最有用的落地点：

- 它能非常直接地支撑你后面“形态先验值得保留”这条主张。
- 即使你完全不做无监督，这篇也能让 related work 更完整。

### 2.6 `Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`

用途定位：

- 近期弱监督腺体分割代表作。
- 适合放在“标注代价高时，image-level weak supervision 也能把 gland segmentation 做到较强表现”的分支里。

可直接提取的关键信息：

1. 摘要说明：作者提出两阶段弱监督框架 `MAA`，只用 `image-level labels`，核心由 `Multi-Level Attention Fusion (MAF)` 和 `Affinity Refinement (AR)` 组成。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.1

2. 论文明确把 gland segmentation 的难点归因于 blurred boundaries、staining inconsistency、overlapping cells 和 densely packed tissue structures。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.1, p.2

3. 数据使用 `GlaS`，切分为训练 `85`、测试 `80`；训练前又把图像裁成 `112×112` patch，overlap 为 `56`，总共得到 `9703` 个训练 patch，并赋予 gland/background 的 image-level labels。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.3, p.4

4. `MAF` 模块从 Transformer 的 `3rd / 5th / 6th` 层融合层级特征，以生成比标准 CAM 更完整、更少噪声的 pseudo-label。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.5, p.7

5. `AR` 模块再基于 pixel affinity 学习语义一致性，进一步改善 pseudo-label 的空间连贯性与边界精度。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.7

6. 在 `GlaS` 上，作者报告 `IoU 81.99% / Dice 90.10%`，比文中对照的 `OEEM` 提高 `4.43%` IoU。
   - 来源：`Weakly_Supervised_Gland_Segmentation_Based_on_Hierarchical_Attention_Fusion_and_Pixel_Affinity_Learning_2025.pdf`, p.1, p.3

对当前实验最有用的落地点：

- 如果你后续数据标注有限，这篇很适合留作升级路线参考。
- 即使当前不做弱监督，这篇也能补近两年腺体分割 related work 的完整性。

### 2.7 `Automatic_Tumor_Grading_on_Colorectal_Cancer_Whole-Slide_Images_2022.pdf`

用途定位：

- 这篇不是 gland segmentation 论文本身，但它把 `gland formation` 和 `CRC prognosis` 直接接起来了。
- 很适合做“为什么 gland formation 值得自动化量化”的下游论据。

可直接提取的关键信息：

1. 论文开头直接指出：WHO grading system 对 CRC 的组织学分级依赖 whole-slide image 上 `gland formation density` 的判断。
   - 来源：`Automatic_Tumor_Grading_on_Colorectal_Cancer_Whole-Slide_Images_2022.pdf`, p.1, p.2

2. 作者构建了两个模型：一个 tissue classifier 把区域分成 glands、stroma、immune cells 等；另一个 gland formation classifier 把区域分成不同 gland formation 程度。
   - 来源：`Automatic_Tumor_Grading_on_Colorectal_Cancer_Whole-Slide_Images_2022.pdf`, p.2, p.3

3. 基于这两类 heatmap，作者定义了 `semi-quantitative gland formation ratio (SGFR)`，并进一步训练 `SPPSN` 生存模型去预测 overall survival。
   - 来源：`Automatic_Tumor_Grading_on_Colorectal_Cancer_Whole-Slide_Images_2022.pdf`, p.3, p.4

4. 摘要给出的结果表明：单独用 `SGFR` 的 survival discrimination 为 `c-index 0.62`，深度 survival grade 为 `0.64`，且后者在多变量 Cox 模型中表现更好。
   - 来源：`Automatic_Tumor_Grading_on_Colorectal_Cancer_Whole-Slide_Images_2022.pdf`, p.1

对当前实验最有用的落地点：

- 这篇特别适合支撑“腺体结构/腺体形成比例不仅影响诊断，也会连接到分级与预后”。
- 以后如果你要把分割结果往更深的预后建模扩展，这篇是很自然的桥接文献。

### 2.8 `Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`

用途定位：

- 一篇把 gland segmentation、cancer detection、grade prediction 串成 end-to-end pipeline 的论文。
- 适合补“分割不是终点，而是整个计算病理流程中的上游模块”。

可直接提取的关键信息：

1. 摘要明确提出一个端到端计算病理 pipeline，覆盖 `gland segmentation`、`cancer detection` 和 `grade prediction` 三个环节。
   - 来源：`Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`, p.1

2. 方法层面，作者不是只做 gland boundary，而是还对 gland 内部结构如 `epithelial cells / nuclei / lumen` 做分割，并结合多尺度特征做后续分类。
   - 来源：`Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`, p.2, p.3

3. 数据使用两套机构数据：`RMC (n=174)` 和 `GlaS (n=165)`；结果中 gland segmentation 达到 `RMC 87.5% / GlaS 88.4%`。
   - 来源：`Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`, p.1

4. 进一步的 malignancy detection 和 grade prediction 结果都很高，摘要给出 `97.6% / 98.3%` 的检测准确率，以及两数据集上均约 `98.6%` 的 grade prediction 表现。
   - 来源：`Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`, p.1

5. 更重要的是 cross-institution generalization：在一套数据上训练、另一套上测试时，恶性检测和 grading 仍保持 `93%+` 和 `95%` 左右表现。
   - 来源：`Segmentation_and_Grade_Prediction_of_Colon_Cancer_Digital_Pathology_Images_Across_Multiple_Institutions_2019.pdf`, p.1

对当前实验最有用的落地点：

- 这篇适合放在 discussion 里，说明 gland segmentation 的价值可以自然延伸到 detection / grading。
- 也能支撑你后面强调跨机构泛化的重要性。

### 2.9 `Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`

用途定位：

- 这篇很适合补“半监督 histopathology segmentation 需要把 stain variation 和 structure variation 分开处理”这条近期路线。
- 对你的价值主要在于提供一个更贴近病理图像特性的 SSL 设计思路，而不是通用语义分割式的一刀切 consistency。

可直接提取的关键信息：

1. 摘要直接指出：作者提出 `Color-Structure Dual-Student (CSDS)`，目标是在标注有限的 histopathology segmentation 中显式 disentangle `stain appearance` 和 `tissue structure` 两类变化来源。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.1

2. 论文动机说得很明确：现有 SSL 方法通常整体处理图像，却忽略了病理图像中“染色差异”和“组织形态变化”是 entangled 但语义不同的两类因素，这会限制鲁棒性与可解释性。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.2

3. `CSDS` 由两个 student 组成：`color student` 用 `histogram matching / color jittering` 学染色变化，`structure student` 用 `elastic deformation` 学形态变化；同时再用一个共享 `EMA teacher` 生成 pseudo-label。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.3, p.5

4. 这篇比较有启发性的一点是它没有只做普通 entropy filtering，而是显式设计了 `color-aware uncertainty` 与 `structure-aware uncertainty`，用来调制 teacher supervision。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.3, p.4

5. 实验覆盖 `GlaS` 和 `CRAG`，实现使用 `DeepLabV3+ + ResNet-101`，输入统一缩放到 `256×256`，优化器为 `AdamW(lr=1e-4)`，batch size 为 `4`。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.5, p.6

6. 结果部分强调：在低标注比例设置下，`CSDS` 在 `GlaS` 上达到 `Dice 82.86 / Jaccard 72.01`，在 `CRAG` 上达到 `Dice 75.25 / Jaccard 63.00`，并持续优于文中比较的其他 SSL 方法。
   - 来源：`Learning_Disentangled_Stain_and_Structural_Representations_2025.pdf`, p.6, p.7

对当前实验最有用的落地点：

- 如果你后续做半监督，这篇很适合支撑“病理图像里的颜色与结构扰动要分治建模”。
- 即使不直接复现，它也能增强你 related work 对 histopathology-specific SSL 设计的理解深度。

### 2.10 `Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`

用途定位：

- 这篇是很直接的 prompt-based gland segmentation 论文，尤其适合补“foundation model / SAM 能不能进入腺体分割”这条线。
- 它最大的价值不只是用了 `SAM`，而是把 `cancer grade` 信息变成 prompt，说明上游病理语义可以反过来帮助分割。

可直接提取的关键信息：

1. 摘要和引言都明确指出：癌症分级依赖 gland morphology 与 gland formation，因此如果模型事先知道 gland 更偏 `benign` 还是 `malignant`，就可能更好地分割腺体。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.1

2. 方法上，作者先用 `ViT` 做 benign/malignant 分类，再用 `Grad-CAM++` 生成 heat map，把这张热图作为 `grade prompt` 送入分割分支。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.2

3. 分割主干采用 `SAM`，并且不是只出 gland mask，还同时预测 `gland` 和 `contour`，以便更好地区分紧邻腺体。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.2, p.3

4. 训练流程分阶段进行：先微调分类分支，再加载 fine-tuned `SAM` 参数初始化分割分支；其中分类分支在 `test A / test B` 上达到 `97.1% / 98.7%` 准确率。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.3

5. 损失函数不是 Dice，而是结合 `U-Net` 风格 weight map 的 `weighted MSE loss`；推理时对 patch overlap 区域做平均，并用 `0.5` 阈值得到二值 mask。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.3

6. 在 `GlaS` benchmark 上，`Prompted SAM-H` 达到 `F1 0.929 / 0.841`、`Object Dice 0.921 / 0.881`、`Object Hausdorff 41.189 / 74.300`（Test A / Test B），优于文中对照的 fine-tuned SAM 版本和多个旧方法。
   - 来源：`Gland_Segmentation_Using_SAM_with_Cancer_Grade_as_a_Prompt_2025.pdf`, p.4

对当前实验最有用的落地点：

- 它很适合放在 related work 里，说明 prompt-based / foundation-model 路线已经开始进入 gland segmentation。
- 如果你后面想把 grading 信息融入分割，这篇是最直接的桥接文献之一。

### 2.11 `Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`

用途定位：

- 这篇代表的是“半监督不仅要做像素一致性，还要做 topology consistency”的路线。
- 它特别适合拿来支撑你后面如果想强调腺体分割天然有拓扑约束，不应只看局部 pixel loss。

可直接提取的关键信息：

1. 摘要直接批评现有 semi-supervised segmentation 多依赖 `pixel-level consistency`，容易传播 noisy pseudo-label，并产生 fragmented 或 topologically invalid masks。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.1

2. 作者提出 `Topology Graph Consistency (TGC)`，核心思想是把预测 mask 和参考 mask 都构造成图，再对齐 `Laplacian spectra`、`component counts` 和 `adjacency statistics`。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.1, p.2

3. 图构造不是在像素层面展开，而是先从 probability map 提取 gland region 的 `centroids`，再建立 `k-nearest neighbor graph`，这使它更接近 object/topology 级约束。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.2

4. 总损失把 `Dice+CE` 和图拓扑损失联合起来，其中 `Lspec`、`Lconn`、`Ladj` 分别约束谱特性、连通分量数量和邻接关系，且无标注阶段的 topology regularization 会逐步 ramp-up。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.2

5. 实现采用两个同架构模型互换 pseudo-label，主干使用 `DeepLabV3+ + ResNet-101`，并在 `GlaS` 与 `CRAG` 上评估。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.2

6. 摘要结论指出：`TGC` 在 `5%-10%` supervision 下达到 state-of-the-art，并显著缩小与 full supervision 的差距；即使这篇篇幅较短，它的 methodological message 很强。
   - 来源：`Graph-Theoretic_Consistency_for_Robust_and_Topology-Aware_Semi-Supervised_Histopathology_Segmentation_2025.pdf`, p.1

对当前实验最有用的落地点：

- 如果你以后走 topology-aware 路线，这篇可以直接作为立论支撑。
- 即使暂时不做半监督，它也能帮助你在 discussion 中说明“腺体 mask 的连通性和结构合法性”为什么值得显式约束。

### 2.12 `PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`

用途定位：

- 这篇适合补“domain-specific pretraining / diffusion representation”这条更新的特征提取路线。
- 它的重点不在于把 diffusion 当生成器，而是把 pathology-specific LDM 当 feature extractor 来做 segmentation。

可直接提取的关键信息：

1. 摘要说明：`PathSegDiff` 利用 pathology-specific `Latent Diffusion Model (LDM)` 作为预训练特征提取器，再接一个简单的全卷积分割头完成病理图像分割。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.1

2. 作者明确强调：相比依赖 ImageNet 或普通监督预训练的 backbone，domain-specific diffusion pretraining 更能编码 histopathology 中复杂而细粒度的 tissue structure。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.1, p.2

3. 方法流程是先把图像切成 non-overlapping patches，用 `SSL encoder` 作为 conditioning 从 LDM U-Net 的 middle block 与 upsampling blocks 抽特征，再上采样、拼接，并送入 `FCN mask generator`。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.4, p.5

4. 评估覆盖 `BCSS` 和 `GlaS`；其中 `GlaS` 使用官方划分 `train 85 / test_A 60 / test_B 20`。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.5

5. 在 `GlaS` 上，`PathSegDiff` 达到 `Accuracy 0.900 / Dice 0.896 / mIoU 0.812 / F1 0.894`，优于 `Res-UNet`、`CUMedVision2`、`MedSegDiff` 以及无条件扩散版本。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.6, p.7

6. 消融还表明：扩散过程的 `early-middle timesteps`（如 `t=50`）提取到的特征更适合分割，而 `blocks 4-6` 等中间层对结构与语义最平衡。
   - 来源：`PathSegDiff_Pathology_Segmentation_Using_Diffusion_Model_Representations_2025.pdf`, p.9

对当前实验最有用的落地点：

- 它很适合用来支撑“病理专用预训练比通用自然图像预训练更值得考虑”。
- 如果你后续想升级 encoder 或做特征蒸馏，这篇是比较新的路线参考。

- 它的意义不只是再做一次 gland segmentation，而是强调腺体不仅要和背景分开，还要彼此分开，才能做后续 morphometrics。

可直接提取的关键信息：

1. 摘要明确指出：`mucous glands` 是消化病理里的重要诊断元素，而结肠息肉恶性潜能评估的第一步就是 gland segmentation。

对当前实验最有用的落地点：

### 2.15 `Inter-_and_Intra-Uncertainty_Feature_Aggregation_for_Semi-Supervised_Histopathology_Segmentation_2024.pdf`

- 这篇非常适合补“复杂、低分化、粘连 gland 该怎么分”的传统-深度混合路线。
- 它最大的价值是把 `lumen`、`rough gland boundary`、`epithelial nucleus` 三类形态约束显式结合起来。

可直接提取的关键信息：

1. 摘要开头就指出：`poorly differentiated colorectal glands` 难以准确分割，因此作者专门提出面向复杂 gland 的分割方案。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.1

2. 方法第一步不是直接分割 gland，而是先用 `U-Net` 做 stain separation，得到 `H-stain / E-stain / background` 三种强度图。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.1, p.3

3. 随后作者在 `background channel` 上分割 `lumens`，并结合 epithelial nuclei 与 rough gland boundary 来逐步锁定 gland contour。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.1, p.4, p.6, p.7

4. 很有代表性的设计是基于 `axis of least inertia (ALI)` 和 `similar triangles` 描述 lumen 与候选 contour 的空间对应关系，再结合距离约束选出最优轮廓。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.6, p.7

5. 作者明确声称两点创新：一是用深度学习做 stain separation 并预测 stain coefficient interval，二是基于内轮廓 `lumen` 与外轮廓 `epithelium` 的多重形态约束来找最优 gland contour。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.3

6. 评估覆盖 `GlaS` 和独立的 colon adenocarcinoma dataset`；作者强调在 malignant cases 中，基于 lumen + rough boundary 的方法能得到更好的 shape similarity，并在独立数据上优于对照方法。
   - 来源：`Multiple_Morphological_Constraints-Based_Complex_Gland_Segmentation_in_Colorectal_Cancer_Pathology_Image_Analysis_2020.pdf`, p.14

对当前实验最有用的落地点：

- 这篇非常能支撑你“复杂腺体分割一定要保留形态约束”的论点。
- 对后续 discussion 来说，它也能帮助解释为什么只靠纯像素学习有时不够。

### 2.18 `Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`

用途定位：

- 这篇代表的是更偏网络结构创新的一条线：用 `Neural ODE` 改造 U-Net。
- 它对你最有用的地方不在于病理学解释，而在于说明 gland segmentation 也有人从 `dynamic receptive field / continuous depth` 角度做过探索。

可直接提取的关键信息：

1. 摘要指出：普通 `U-Net` 的一个问题是 receptive field 被硬编码，想扩大感受野通常意味着更多参数和更高内存代价。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.1

2. 作者把 `Neural Ordinary Differential Equations (NODE)` 引入 U-Net，提出 `U-Node`，目标是在保持较低参数量的同时获得动态 receptive field。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.1, p.2

3. 文中明确说：`U-Net` 约有 `30M` 参数，而 `U-Node` 和 `U-ResNet` 仅约 `2M` 参数，显示 NODE 路线在内存和参数效率上的潜力。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.1, p.3

4. 实验在 `GlaS` 上进行，训练使用 `Adam`、batch size `8`、`600 epochs`，并在测试时使用翻转 test-time augmentation。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.3

5. 结果表明：`U-Node` 达到 `Object Dice 0.881 / F1 0.861 / Hausdorff 59.5`，优于作者实现的普通 `U-Net`（`0.868 / 0.841 / 69.6`）和 `U-ResNet`。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.3

6. 作者讨论中认为：NODE 的优势主要来自连续深度建模与更灵活的感受野，但也指出推理中需要多次 function evaluations，是一个速度与结构灵活性的权衡。
   - 来源：`Neural_Ordinary_Differential_Equations_for_Semantic_Segmentation_of_Individual_Colon_Glands_2019.pdf`, p.2, p.3

对当前实验最有用的落地点：

- 如果你后面要补“非常规 backbone / receptive field 设计”这条 related work，这篇值得一提。
- 但从你的主线任务看，它更像结构探索型参考，不一定是最高优先级复现对象。

### 2.19 `One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`

用途定位：

- 这篇是很强的多任务数字病理论文，适合补“一个模型同时做 gland / lumen / nuclei / tissue type”这条系统化路线。
- 对你最有价值的不是直接替代主模型，而是说明 gland segmentation 可以自然嵌进更大的组织学分析管线。

可直接提取的关键信息：

1. 摘要明确指出：作者提出 `Cerberus` 多任务模型，同时完成 `nuclei`、`glands`、`lumina` 和 `tissue regions` 的分割/分类，并利用多源数据共享表示。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.1

2. 论文强调一个重要前提：多任务要想“同时预测”有意义，各任务的输入必须在 tissue type、stain 和 resolution 上 `aligned`，否则只能做泛化 encoder，不能真正共预测。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.2, p.16

3. `Cerberus` 采用 `ResNet34` 共享 encoder，加多个 task-specific decoders；分割分支用 `U-Net style decoder`，分类分支用 `global average pooling + FC layers`。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.5

4. 数据规模非常大：作者汇总 `51,157 glands`、`56,358 lumina`、`495,179 nuclei`，以及数十万 tissue-type patches，并处理了 `599` 张 TCGA colorectal WSIs。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.1, p.8, p.9, p.16

5. 在 gland / lumen 分割上，`Cerberus+` 外部测试达到 `Gland PQ 0.650`、`Lumen PQ 0.530`，优于 `U-Net`、`MILD-Net` 和 `Rota-Net` 等对照。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.10

6. 论文还显示：引入 `mixed batch MTL` 与附加的 patch classification 任务后，外部测试泛化更好；同时用 `Cerberus` 预训练权重还能提升 signet ring cell detection 等下游任务。
   - 来源：`One_Model_is_All_You_Need_Multi-Task_Learning_Enables_Simultaneous_Histology_Image_Segmentation_and_Classification_2022.pdf`, p.11, p.12, p.15

对当前实验最有用的落地点：

- 它很适合支撑你把 gland segmentation 放进更大的一体化病理分析框架。
- 如果后面你想做多任务联合学习或把 lumen 一起纳入，这篇是很关键的参考。

### 2.20 `Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`

用途定位：

- 这篇是 gland segmentation 半监督里的代表作之一，应该保留。
- 它的核心不是普通 consistency，而是用 `pairwise relation` 去学图像间语义一致性。

可直接提取的关键信息：

1. 摘要说明：作者提出 `PRS2`，由 `segmentation network (S-Net)` 与 `pairwise relation network (PR-Net)` 组成，用于半监督 gland segmentation。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.1

2. `S-Net` 采用 `DeepLabv3+` 作为主分割网络，而 `PR-Net` 则在无监督条件下利用图像对之间的语义一致性来增强表示学习；两者共享 encoder。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.3, p.4, p.5

3. 论文专门把 `object-level Dice` 从评测指标改造成训练损失，并与 `cross-entropy`、`global Dice` 联合，目的就是更好处理 `touching glands`。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.3, p.4

4. 数据实验同时覆盖 `GlaS` 与 `CRAG`；在一个数据集上评估时，另一个数据集的训练集可以充当 unlabeled data，这个设置很适合病理场景的半监督扩展。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.6

5. 结果上，`PRS2` 在 `GlaS` 上达到 `Obj-D 90.6 / Obj-F 89.0 / Obj-H 55.1`，在 `CRAG` 上达到 `Obj-D 89.2 / Obj-F 84.3 / Obj-H 113.1`，总体优于文中多个对照方法。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.7

6. 更重要的是，作者显示：只用 `50%` 标注图像训练的 `PRS2`，效果接近 fully-supervised `S-Net` 用 `100%` 标注训练的结果，说明 unlabeled histology data 确实能替代近一半 dense labels。
   - 来源：`Pairwise_Relation_Learning_for_Semi-supervised_Gland_Segmentation_2020.pdf`, p.8, p.9

对当前实验最有用的落地点：

- 这是你写半监督 gland segmentation related work 时几乎不能漏的一篇。
- 如果以后要做关系建模、图像对一致性或无标注跨数据集迁移，这篇都很有参考价值。

### 2.21 `Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`

用途定位：

- 这篇是 gland+lumen 联合分割里很有代表性的 `rotation equivariant` 工作。
- 它对你很有用，因为病理图像天然存在旋转对称性，而普通 CNN 往往没有显式利用这一点。

可直接提取的关键信息：

1. 摘要指出：gland 和 lumen 的形态分析能反映恶性程度，而现有自动分割通常没有考虑 histology images 内在的 `rotational symmetry`。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.1

2. 作者采用 `group equivariant convolutions`，具体使用 `p4` 群，也就是以 `90°` 倍数旋转构成的对称群，让网络对旋转呈现可预测的 equivariance。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.1, p.3, p.4

3. 网络是 dual-branch encoder-decoder：一个分支做 `gland segmentation`，一个分支做 `lumen segmentation`；其中 gland 分支采用 `multi-class strategy` 帮助分开 touching instances。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.1, p.5

4. 在 `GlaS` 上，Rota-Net 的 gland 分割达到 `F1 0.920/0.824`、`Object Dice 0.919/0.849`、`Object Hausdorff 40.99/95.72`（Test A/B），拿到最优加权 rank。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.6

5. 同时做 gland+lumen 分割时，在 Test A 上 Rota-Net 达到 `Gland F1 0.920 / Dice 0.919` 与 `Lumen F1 0.831 / Dice 0.824`，明显优于对照的 U-Net 和 FCN-8。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.7

6. 消融表明：rotation equivariant 设计本身就带来显著收益，而在 gland 输出端再加 contour-aware 的多分类策略还能继续提升分离效果。
   - 来源：`Rota-Net_Rotation_Equivariant_Network_for_Simultaneous_Gland_and_Lumen_Segmentation_in_Colon_Histology_Images_2019.pdf`, p.7

对当前实验最有用的落地点：

- 如果你后面考虑联合分割 gland 和 lumen，这篇是优先级很高的对照。
- 它也能帮助你论证“旋转对称性是病理图像里可以显式利用的归纳偏置”。

### 2.22 `Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`

用途定位：

- 这篇是更泛化的病理图像半监督分割论文，不是专做 gland，但方法设计对 gland 非常可借鉴。
- 它特别适合补“多分支注意力 + 软标签蒸馏”这条半监督路线。

可直接提取的关键信息：

1. 摘要说明：作者提出 `CDMA`，即 `Cross Distillation of Multiple Attentions`，用于半监督病理图像分割。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.1

2. 主体网络 `MTNet` 是一个 `tri-branch network`，三个 decoder 分别使用 `channel attention (CA)`、`spatial attention (SA)` 和 `CSA`，目的是生成多样化、互补的预测。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.1, p.3, p.4

3. 这篇的重要点在于没有用硬 pseudo labels，而是提出 `Cross Decoder Knowledge Distillation (CDKD)`，让每个分支学习另两个分支的 `soft predictions`，以减少 noisy pseudo labels 带来的过拟合。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.3, p.5

4. 作者还对三个分支的平均预测施加 `uncertainty minimization`，从而同时增强 inter-branch consistency 与低熵预测。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.3, p.6

5. 实验在 `DigestPath` 结肠病理病灶分割数据集上进行，`5%/10%` 标注条件下，`CDMA` 的 `DSC` 分别达到 `69.72% / 72.24%`，优于文中八种 SSL 方法。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.6, p.7, p.8

6. 消融实验说明：`CDKD + uncertainty minimization + 三种不同 attention branches` 的组合确实优于仅双分支、无 attention 或直接 argmax pseudo labels 的变体。
   - 来源：`Semi-supervised_Pathological_Image_Segmentation_via_Cross_Distillation_of_Multiple_Attentions_2023.pdf`, p.8, p.9

对当前实验最有用的落地点：

- 它适合补你对“半监督病理分割不一定只能做 mean-teacher”的理解。
- 如果你以后做多分支 decoder、attention diversity 或 soft-label distillation，这篇可以直接参考。

### 2.23 `To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`

用途定位：

- 这篇非常适合补“domain-specific pretraining 到底值不值得”这个经常被默认却未必总成立的问题。
- 它对你最有帮助的地方是：把 gland segmentation 和 cell segmentation 分开看，而不是笼统说 pathology pretraining 一定更好。

可直接提取的关键信息：

1. 摘要开门见山：作者专门比较 `domain-specific` 与 `non-domain-specific` 预训练权重对 `gland` 和 `cell segmentation` 的影响，并分析 dataset size 与 OOD testing。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.1

2. 研究对象包括 `GlaS`、`CRAG` 两个 gland 数据集，以及 `KUMAR`、`CPM17`、`TNBC` 三个 cell segmentation 数据集。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.4

3. 他们统一用 `U-Net + ResNet50 backbone`，比较 `random initialization`、`ImageNet supervised`、`ImageNet SSL` 和 `SSLPathology` 四类初始化。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.4, p.5

4. 论文的核心结论之一是：在 `gland segmentation` 上，尤其是训练数据较少时，`domain-specific pretraining` 确实带来显著优势；但随着数据增多，这个优势会减弱。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.5, p.6

5. 与之相对，`cell segmentation` 并没有稳定显示 pathology-specific pretraining 的额外收益，说明“病理预训练有用”不是对所有任务都成立。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.6

6. OOD 实验还表明：domain-specific pretraining 通常能减小一定的跨数据集性能下降，但并不能根本解决 site bias / distribution shift；作者明确说其收益并非 universal。
   - 来源：`To_Pretrain_or_Not_to_Pretrain_A_Case_Study_of_Domain-Specific_Pretraining_for_Histopathology_Segmentation_2023.pdf`, p.7, p.8, p.9

对当前实验最有用的落地点：

- 这篇能帮助你更谨慎地写“病理专用预训练”的结论，避免过度泛化。
- 如果你后面想做 encoder 初始化选择，它是非常直接的决策参考。

### 2.24 `Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`

用途定位：

- 这篇就是 `RepLKNet` 原始论文，是你当前“大核卷积”路线里最值得保留的现代代表作。
- 它不直接针对腺体分割，但对你解释为什么要在 CNN 里重新重视大核、感受野和形态偏置非常关键。

可直接提取的关键信息：

1. 摘要明确指出：作者重新审视现代 CNN 中的大核设计，并提出 `RepLKNet`，其卷积核可以大到 `31x31`。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1

2. 论文的核心立场是：在视觉 Transformer 兴起之后，使用少量大卷积核而不是堆叠大量小卷积，可能是更强的 CNN 设计范式。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1

3. 作者提出了 `five guidelines` 来构建高性能大核 CNN，其中一个代表性做法是 `re-parameterized large depth-wise convolutions`。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1

4. 摘要结果里直接给出：`RepLKNet` 在 ImageNet 与多个下游任务上显著缩小了 CNN 和 ViT 的差距，并在 `ADE20K` 上达到 `56.0% mIoU`。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1

5. 论文还强调：与小核 CNN 相比，大核 CNN 具有更大的 `effective receptive fields`，这正好能支撑你现在的感受野动机。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1, p.2

6. 另一条很适合写动机的话是：作者认为大核 CNN 表现出更高的 `shape bias rather than texture bias`，这对病理图像里的结构建模很有启发。
   - 来源：`Scaling_Up_Your_Kernels_to_31x31_Revisiting_Large_Kernel_Design_in_CNNs_2022.pdf`, p.1

对当前实验最有用的落地点：

- 这篇非常适合放在你 `LKMA` 或大核模块动机的第一梯队证据里。
- 如果你后面要解释“为什么不是只引用 GCN/Wavelet，而是还要补一个现代大核 CNN 代表”，这篇就是最稳的答案。
