### 2.1 `A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`

用途定位：

- 这篇是 `BSDS / human segmented natural images` 数据库的源头论文之一，最适合补你这里 segmentation benchmark 与人工标注 ground truth 的最原始出处。
- 它不直接决定腺体分割模型结构，但对解释“为什么分割算法可以和多人类标注结果做量化比较”非常关键。

可直接提取的关键信息：

1. 摘要直接说明：作者构建了一个由人类产生 `groundtruth segmentations` 的自然图像数据库，并把它用于 `evaluating segmentation algorithms` 与 `measuring ecological statistics`。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.2

2. 论文的核心立场很重要：图像分割并不一定只能在具体下游任务里评估，也可以通过与 `multiple human observers` 的分割结果比较来直接评价，而且不同人的分割在同一图像上具有相当高的一致性。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.3

3. 文中给出了数据库建设口径：作者从 `Corel` 图像库里选取了 `1000` 张具有代表性的 `481x321 RGB` 自然图像，作为后续 benchmark 的基础样本。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.4

4. 这篇还给出一个很实用的数据库进展说明：论文当时已收集到 `3000` 个分割、`25` 个参与者、`800` 张图像，并计划最终达到 `1000` 张图像各至少 `4` 份灰度和 `4` 份彩色分割。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.5

5. 在评价方法上，作者提出了 `GCE` 和 `LCE` 两种 segmentation consistency / error measure，并强调它们对 `refinement` 具有容忍性，因此更适合比较不同粒度的人类或算法分割。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.6-p.7

6. 论文还把该数据库用于 `NCuts` benchmark，并给出一个清晰参照：`NCuts` 的平均误差约为 `22%` (`LCE`) 和 `28%` (`GCE`)，而人类之间约为 `7%` 和 `11%`，说明该数据集能够把算法和人类表现拉开。
   - 来源：`A_Database_of_Human_Segmented_Natural_Images_and_its_Application_to_Evaluating_Segmentation_Algorithms_2001.pdf`, p.8

对当前实验最有用的落地点：

- 这篇最适合支撑你在 related work 或指标说明里写清：经典分割 benchmark 并不是随便拿人工标注当参照，而是有一整套 human consistency 论证链。
- 如果你后面要解释 `BSDS`、`Boundary F1`、人类边界标注与算法评价之间的关系，这篇是很稳的上游出处。

### 2.2 `A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`

用途定位：

- 这篇不是 CRC 专题论文，但非常适合充当你后面写方法细节时的“损失函数选择说明书”。
- 它的价值主要在于帮你更系统地解释：为什么不同数据分布、边界难度和类别不平衡会对应不同 loss 设计。

可直接提取的关键信息：

1. 摘要明确说：这篇综述总结了近年 semantic segmentation 中常见 loss function 的使用场景，重点关注 `biased data`、`sparse segmentation` 等不同训练条件。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.1

2. 论文系统梳理了约 `15` 类常见 loss，包括 `Binary Cross-Entropy`、`Weighted/ Balanced Cross-Entropy`、`Focal`、`Dice`、`Tversky`、`Focal Tversky`、`Shape-aware`、`Hausdorff Distance loss`、`Combo loss` 等。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.2, p.3, p.4

3. 对你最实用的一条是：`Binary Cross-Entropy` 更适合相对 balanced 的数据，而 `Weighted BCE`、`Balanced CE` 和 `Focal loss` 更适合 skewed / highly-imbalanced segmentation。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.2, p.5

4. 文中还强调：`Tversky` 和 `Focal Tversky` 适合更关注 hard examples 或小目标 ROI 的情形；`Shape-aware loss` 与 `Hausdorff-based loss` 则更适合 hard-to-segment boundaries 和结构形状约束。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.3, p.4, p.5

5. 结论部分明确指出：不存在一个对所有数据都最优的 universal loss，loss 选择应依赖数据分布、类别偏斜、边界复杂度等任务属性。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.6

6. 该文自己的实验结论也提到：`Focal Tversky loss` 与 `Log-Cosh Dice loss` 在其测试中表现最好，说明 focus-based loss 往往更适合高不平衡场景。
   - 来源：`A_Survey_of_Loss_Functions_for_Semantic_Segmentation_2020.pdf`, p.1, p.6

对当前实验最有用的落地点：

- 如果你后面要解释为什么不只用单一 Dice，这篇非常好用。
- 它也适合支撑你在腺体边界复杂、前景比例不稳定时引入 hybrid loss 的合理性。

### 2.3 `Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`

用途定位：

- 这篇是边界检测和边界评价 benchmark 的经典论文，最适合补你这里 `Boundary F1 / precision-recall` 的定义来源。
- 它不直接决定腺体分割模型怎么搭，但对写指标出处、边界评价口径和 related work 里的 boundary benchmark 很有价值。

可直接提取的关键信息：

1. 论文首页就明确说明：作者同时研究 `contour detection` 和 `image segmentation`，并把分割评价统一到 contour/boundary 视角下。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.1

2. 论文明确写到：在 `BSDS300` 上，算法评价采用 `precision-recall framework`，这是你后面引用 `Boundary F1` 时最关键的 benchmark 口径之一。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.1

3. 文中还直接说明：该 benchmark 通过把 `machine generated contours` 与 `human ground-truth` 做比较来评估算法表现。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.1

4. 一个对你最有用的细节是：作者允许把 `region boundaries` 也按 contours 来评价，这使它不仅能用于纯边界检测，也能支撑分割结果的边界视角评估。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.1

5. 图 1 的说明里明确把方法按 `maximum F-measure` 排序，并给出 `2PrecisionRecall / (Precision + Recall)` 的定义，这正好能托住你后面说的 `Boundary F1 / Boundary F-score`。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.2

6. 论文整体结论是：`gPb` 和 `gPb-owt-ucm` 在 contour detection 与 segmentation benchmark 上都显著优于当时主要对照方法，因此这篇既是指标来源，也是经典 benchmark 论文。
   - 来源：`Contour_Detection_and_Hierarchical_Image_Segmentation_2011.pdf`, p.1, p.2

对当前实验最有用的落地点：

- 这篇最适合在方法或实验部分补一句：边界相关评价可追溯到 `BSDS` 的 `precision-recall / maximum F-measure` 口径。
- 如果你后面要解释 `Boundary F1` 不是临时自造指标，而是经典边界 benchmark 的延续，这篇就是最稳的出处之一。

### 2.4 `Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`

用途定位：

- 这篇是自然图像边界检测的经典论文，最适合补你这里 `brightness + texture` 联合建模和 `precision-recall / F-measure` 边界评价链条。
- 它不属于腺体分割主线方法论文，但非常适合作为 boundary benchmark 发展线上的关键节点。

可直接提取的关键信息：

1. 摘要直接说明：论文目标是利用局部图像测量准确检测并定位自然场景中的边界，作者构造了同时响应 `brightness` 与 `texture` 变化的特征，并用人类标注图像训练分类器，最终通过 `precision-recall curves` 证明方法优于已有方案。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.1

2. 文中对任务本身的定义也很关键：作者希望估计“某个像素中心是否有 boundary 穿过”的 `posterior probability`，并明确指出这种 `local boundary model` 是更高层 segmentation 算法的重要组成部分。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.1

3. 论文明确批评了只看亮度突变的传统做法：`Canny` 这类 brightness edge 模型在纹理区域内部会产生大量错误响应，同时又容易漏掉平均亮度变化很小但纹理不同的真实边界。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.1

4. 该文的一个核心贡献是把 cue combination 写成监督学习问题：使用由多个人类受试者手工分割得到的大规模自然图像数据，把每个像素建模为 `on-boundary / off-boundary` 分类任务。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.2

5. 方法层面，作者把 `oriented energy (OE)` 和 `texture gradient (TG)` 在多尺度上联合起来，并通过 localization 过程显著提升边界定位质量，尤其对 `texture gradient` 的提升很明显。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.3-p.5

6. 评价层面，论文继续采用 `precision-recall` 与 `F-measure`；图 4 给出的代表结果是 `Human F=.75`、`Us F=.67`、`Nitzberg F=.65`、`Canny F=.57`，说明学习式亮度-纹理联合检测明显优于经典基线，但仍低于人类水平。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.6-p.7

7. 结论进一步强调：该方法实际上是在每个位置和方向上建模 boundary 的真实 `posterior probability`，因此对更高层视觉算法尤其有用。
   - 来源：`Learning_to_Detect_Natural_Image_Boundaries_Using_Brightness_and_Texture_2002.pdf`, p.8

对当前实验最有用的落地点：

- 这篇非常适合用来说明：边界检测不能只看灰度突变，纹理和亮度联合建模是经典而成熟的路线。
- 如果你后面要把 `Boundary F1 / precision-recall` 放回经典自然图像 benchmark 脉络，这篇和前面的 `Contour Detection`、`A Database ...` 能形成很完整的链条。

### 2.5 `On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`

用途定位：

- 这篇不是病理专文，但很适合用来补“轻量化分割网络怎么做”的方法背景。
- 对你最有用的是它把 `效率-精度` 权衡讲得很清楚，能帮助你后面解释为什么不一定一味追求最大模型。

可直接提取的关键信息：

1. 摘要明确指出：很多高性能 semantic segmentation 模型过于复杂，不适合低内存、低时延场景，因此需要研究 `compact and efficient models`。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.1

2. 该综述把实时分割方法分为五类 taxonomy：`encoder-decoder`、`multi-branch`、`meta-learning`、`attention` 和 `training pipeline`。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.2, p.6

3. 方法层面总结了一系列经典提速手段：`early downsampling`、`lightweight decoder`、`depthwise-separable convolution`、`grouped convolution`、`asymmetric convolution`、`bottleneck` 等。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.4, p.5

4. 文章特别强调：实时分割始终面对一个稳定存在的 `latency-accuracy trade-off`，即更快往往意味着更低精度，尤其在边界和细节保留上更明显。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.1, p.15, p.16

5. 在医学图像等应用中，虽然延迟不像自动驾驶那样关键，但如果模型要直接部署在诊断设备上，`memory constraints` 仍然很重要。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.3

6. 结论部分提出一个很适合迁移到腺体分割的判断：高效模型最难的地方是同时融合 `global context` 和 `fine-grained spatial detail`，而边界细节往往是轻量化里最容易丢掉的部分。
   - 来源：`On_Efficient_Real-Time_Semantic_Segmentation_A_Survey_2022.pdf`, p.16

对当前实验最有用的落地点：

- 如果你后面需要为轻量化、低资源部署或快速推理做铺垫，这篇很好用。
- 它也能帮助你解释为什么 gland boundary 细节任务里，过度压缩模型往往会带来代价。

### 2.6 `Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`

用途定位：

- 这篇最适合做 `BDE` 的说明文和补链条文献，不是你方法主线论文，但对指标出处补强很实用。
- 它的价值在于把常见分割评价指标放在一篇文章里系统说明，并明确把 `BDE` 追溯到 `Freixenet et al. (2002)`。

可直接提取的关键信息：

1. 论文定位很清楚：它讨论的是 active contour segmentation 的性能评价问题，目标是建立更系统的 assessment tool。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.1

2. 这篇对你最关键的一句是：`Boundary displacement error (BDE) is proposed by Freixenet et al. (2002)`，这正好能把 `BDE` 的来源往前追溯。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.11

3. 文中还明确解释了 `BDE` 的含义：它衡量的是两个分割边界之间，边界像素到另一边界最近像素的平均位移误差。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.11

4. 这篇并不是只提一个指标，而是把 overlap、region-based 和 boundary-based 多类指标放在一起讨论，因此很适合你后面写“为什么不同任务要看不同评价维度”。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.9-p.11

5. 论文的问题意识也很适合引用：作者强调，单看某一个指标并不足以全面判断分割算法优劣，因此评价应结合不同 aspect 来看。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.1, p.12

6. 对你当前项目来说，这篇最适合放在“如果正文或补充材料里要解释 BDE 是什么”这一层，而不是拿来替代 `GlaS/TA-Net/MILD-Net` 的主评价口径。
   - 来源：`Performance_metrics_for_active_contour_models_in_image_segmentation_2011.pdf`, p.1, p.11, p.12

对当前实验最有用的落地点：

- 如果你后面需要一句更稳的话，可以写成：`BDE` 常被用作边界位移误差指标，其说明性出处可见该文，并进一步追溯到 `Freixenet et al. (2002)`。
- 它适合保留在辅助层，作为边界指标补充说明，而不是提升到主战文献层。

### 2.7 `Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`

用途定位：

- 这篇适合补 `ViT / Transformer segmentation` 的总览背景。
- 它最大的价值是帮你快速定位哪些 Transformer 路线真正适合 dense prediction，而不是把分类模型直接搬过来。

可直接提取的关键信息：

1. 摘要指出：ViT 在分类上成功后逐步进入 semantic segmentation，但原始 `ViT` 由于 patch partitioning 等原因，不能直接作为 dense prediction 的通用 backbone。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.1

2. 论文明确写到：相比 CNN，ViT 的核心优势是 `self-attention` 能建模 `long-range dependencies`；但其代价是通常更依赖大规模数据和更高算力。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.2

3. 文章专门提到医学图像里 ViT 已被用于 `TransUNet`、`Swin-Unet` 等结构，说明 Transformer 路线已经进入 medical segmentation。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.7

4. 对于数据稀缺问题，综述把 `self-supervised learning` 作为重要补救方向，指出 segmentation 可以把 SSL 训练得到的权重迁移到下游 dense prediction。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.8, p.9, p.10

5. 文中重点梳理了 `SETR`、`Swin Transformer`、`Segmenter`、`SegFormer`、`PVT/PVTv2`、`Twins`、`DPT`、`HRFormer`、`Mask2Former` 等主要分支及其 segmentation 设计要点。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.14-p.25

6. 综述结论认为：ViT 很有潜力逐步替代传统 CNN，但当前仍受制于高计算代价，因此未来方向之一是让 Transformer 更 `lightweight and efficient`。
   - 来源：`Semantic_Segmentation_Using_Vision_Transformers_A_Survey_2023.pdf`, p.26

对当前实验最有用的落地点：

- 如果你后面要在方法背景里解释为什么考虑 Transformer / hybrid encoder，这篇是很好用的上位综述。
- 它也能支持你把 `SegFormer`、`Swin` 这类结构更合理地放进 related work。

### 2.8 `Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`

用途定位：

- 这篇综述最适合补“区域信息和边界信息为什么要协同”的老线索，能把你后面关于 region/boundary 融合的说法放进更长的历史脉络里。
- 它不是具体 benchmark 定义文，但对写 related work、说明边界与区域融合并非临时拼接思路很有帮助。

可直接提取的关键信息：

1. 摘要开门见山指出：单独基于 `boundary` 或单独基于 `region` 的分割方法往往都难以产生准确结果，因此研究逐渐转向利用二者互补性的融合方法；作者综述并归纳出 `7` 类融合策略，还做了真实定量比较。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.1

2. 论文首先按融合时机把方法分成两大类：`embedded integration` 和 `post-processing integration`，前者是在 region 检测过程中直接引入边界信息，后者是在两套结果都得到之后再做融合修正。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.2

3. 在 `embedded integration` 内部，作者又明确区分为两类：一类用边界信息控制 region growing 的 `decision criterion`，另一类用边界信息指导 `seed placement`。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.3-p.5

4. 在 `post-processing integration` 部分，论文把方法进一步归纳为 `over-segmentation`、`boundary refinement` 和 `selection-evaluation` 三种思路，这个分类对你后面整理 related work 很实用。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.5-p.6

5. 文中对 `boundary refinement` 的说明很值得保留：region-based segmentation 往往能较好找到真实区域，但在噪声作用下边界容易非常不规则，因此需要结合 salient edge information 做进一步精化。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.7

6. 这篇还明确提到：region result 作为 snake 初始化、再用动态轮廓做边界精化，是 integrated methods 的一个常见方案；并特别指出在 `MRI` 这类医学图像中，准确分割对诊断非常关键，因此此类方法尤其有意义。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.8

7. 在评价方案上，作者同时给出 `boundary-based evaluation` 和 `region-based evaluation`：前者基于边界点集之间的距离分布签名，后者基于 segmentation 之间的 `directional Hamming distance` 与归一化性能度量。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.10-p.11

8. 结果分析部分指出：从边界精度角度看，`boundary refinement` 策略表现最好，其中基于 snake 能量最小化的 `A6` 在边界精度上尤其突出；但同时还要考虑计算代价与实际平衡。
   - 来源：`Yet_Another_Survey_on_Image_Segmentation_Region_and_Boundary_Information_Integration_2002.pdf`, p.13

对当前实验最有用的落地点：

- 这篇很适合支撑你在综述或讨论里写一句：`region` 和 `boundary` 的融合是图像分割里的长期主题，并且早期就已经形成了较系统的分类和评价框架。
- 如果后面你要强调腺体分割里边界约束、区域一致性和后处理精化并不是彼此割裂的思路，这篇可以作为很稳的辅助文献。

### 2.9 `Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`

用途定位：

- 这篇适合补 `adoptive cell therapy (ACT)` 在 CRC 中的发展背景。
- 它的价值在于帮你把 CRC 免疫治疗从 `ICI` 扩展到 `DC vaccine / CAR-T / CAR-NK / TCR-T / CAR-M` 等更宽的框架。

可直接提取的关键信息：

1. 摘要明确指出：该综述系统梳理了 CRC 中多类免疫细胞治疗，包括 `DC vaccine`、`CAR/TCR-T cells`、`CAR-NK` 等，并强调它们作为 advanced CRC 新型治疗策略的潜力。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.1

2. 引言给出一个很实用的临床背景：advanced CRC 的 `5-year survival` 仍只有约 `20%-30%`，这正是 ACT 受到重视的原因。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.2

3. 文中总结：`CAR-T` 在血液肿瘤中成功，但在 CRC 这类实体瘤里仍受限于 `solid tumor microenvironment`，尤其是 cell infiltration、persistence 和免疫抑制问题。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.1, p.5, p.11, p.12

4. 相比之下，`CAR-NK` 被认为更有吸引力，因为它 `independent of MHC`、`lower risk of GVHD/CRS`，而且不一定需要完全个体化制备。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.1, p.4, p.6

5. 该综述还系统列出多个 CRC 靶点：如 `CEA` 在约 `50%-98.8%` CRC 中高表达，`GUCY2C` 在近 `95%` CRC 中表达，`HER2` 在 CRC 中约 `2%-11%` 过表达，`EpCAM` 则在 `>90%` 肿瘤细胞中高表达。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.5, p.8, p.9, p.10

6. 文章最后强调：要提升 CRC 中 ACT 的实际效果，关键仍是克服免疫抑制性 TME，因此需要 `genetic modification`、`checkpoint inhibitors` 联合和 stromal remodeling 等策略。
   - 来源：`Adoptive_Immune_Cell_Therapy_for_Colorectal_Cancer_2025.pdf`, p.11, p.12

对当前实验最有用的落地点：

- 这篇适合在免疫治疗相关背景里作为“更广义细胞治疗”补充，而不是只写 ICI。
- 如果后面要写 CRC 特定靶点与未来下游治疗方向，这篇很有用。

### 2.10 `An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`

用途定位：

- 这篇适合补 `inflammation-related subtype`、TME 与预后分层。
- 它的亮点是说明“免疫浸润高”并不必然等于“预后更好”，关键要看浸润成分和免疫抑制状态。

可直接提取的关键信息：

1. 摘要说明：作者基于 `inflammation-related genes` 对 CRC 做分型，比较不同亚型的 clinicopathological features、immune infiltration、突变谱、survival 和 drug sensitivity。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.1

2. `NMF` 分型在 `534` 例患者中识别出两个稳定亚型：`INF-low` 和 `INF-high`。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.7

3. 一个很关键的结果是：`INF-high` 虽然炎症和免疫浸润更高，但总体预后更差，`OS/PFS/DSS/DFS` 都更短。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.7, p.14, p.15

4. 文章解释这种悖论的原因：`INF-high` 里富集的并不只是抗肿瘤免疫，还包括 `Tregs`、macrophages、mast cells 等免疫抑制成分，以及更高的 `TIDE` 和 T-cell exhaustion 倾向。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.8, p.15, p.16, p.17

5. 免疫检查点层面，`INF-high` 中 `BTLA`、`CTLA-4`、`LAG-3` 等表达更高，提示更强的 immune evasion 倾向。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.8

6. 治疗敏感性方面，`INF-high` 对 `5-FU`、`Camptothecin`、`Irinotecan`、`Docetaxel` 更敏感，而 `INF-low` 可能更适合免疫治疗。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.1, p.9, p.17

7. 作者还进一步建立了一个 `11-gene` 炎症风险模型，且 `IRRS` 在多变量分析中与年龄、TNM 一起构成独立预后因子。
   - 来源：`An_Inflammation-Related_Subtype_Classification_for_Clinical_Prognosis_in_CRC_2024.pdf`, p.10, p.11, p.12

对当前实验最有用的落地点：

- 这篇很适合放在 TME/预后讨论中，提醒不能把“炎症多”机械等同于“免疫好”。
- 如果后面要解释为什么某些高浸润样本仍然临床更差，它非常有价值。

### 2.11 `Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`

用途定位：

- 这篇适合补 `ADC` 在 CRC 中作为免疫调节工具的背景。
- 它的核心价值不是传统药理综述，而是把 `ADC` 重新解释为把冷肿瘤转热的 `immuno-oncology agent`。

可直接提取的关键信息：

1. 摘要开门见山指出：尤其对免疫学上 `cold` 的 `MSS CRC`，`ADC` 有望超越“定向化疗”的旧定位，成为新一代免疫肿瘤学工具。
   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.1

2. 文章最重要的机制判断是：通过选择 `topoisomerase inhibitors`、`auristatins` 等 payload，ADC 可以诱导 `ICD` 或 `pyroptosis`，从而像 `in situ vaccine` 一样把 TME 从冷转热。
   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.1, p.2, p.3, p.4

   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.1, p.5, p.6, p.7, p.9

4. 具体到 CRC 数据，`CEACAM5` 靶向的 `labetuzumab govitecan` 在 heavily pretreated mCRC 中报道约 `29%` clinical benefit rate；`c-Met` ADC 在高表达肿瘤里 `ORR` 可到 `37.5%`；肿瘤选择性 `EpCAM` ADC 早期数据 `ORR` 约 `28%-43%`。
   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.5, p.6

5. 文章也明确指出 CRC 的现实障碍：target heterogeneity、internalization 效率不足、desmoplastic stroma 与 penetration barrier，这些都限制 ADC 真正打到肿瘤深部。
   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.10, p.11

6. 对免疫治疗背景最重要的一点是：`MSI-H` 仅约 `15%`，而 `MSS` 约 `85%`，后者通常 T-cell infiltration 少、免疫抑制强，因此特别需要 ADC 这类 immune-priming 工具。
   - 来源：`Antibody-Drug_Conjugates_as_Immuno-Oncology_Agents_in_Colorectal_Cancer_Targets_Payloads_and_Therapeutic_Synergies_2025.pdf`, p.11

对当前实验最有用的落地点：

- 这篇适合在“如何把 MSS/冷肿瘤重新激活”这一段里引用。
- 如果后面想把结构病理、药物递送和免疫反应串成更完整的临床意义链，这篇很好用。

### 2.12 `Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`

用途定位：

- 这是很适合放在引言前半段和病理背景部分的一篇基础文献。
- 它能帮助你把“腺体分割为什么重要”说得更符合病理学语言，而不是只用计算机视觉语言。

可直接提取的关键信息：

1. 论文摘要直接指出：CRC 的分类和分期依赖 lesion 的 histological characteristics 和可预测的 morphology changes，而最常见类型是由 adenomatous polyp 进展而来的 adenocarcinoma。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.1

2. 正文明确强调：大多数 CRC 起始于 polyp，早期 detection 和 removal 对改善 prognosis 很关键；这也说明对 gland / polyp 相关结构的观察在病理实践中地位很高。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.1

3. 论文对正常结肠腺体的描述很关键：normal colon glands 是由单层 columnar cells 与大量 goblet cells 构成的单管状 invagination，这为后面理解 dysplasia 和 gland architecture disruption 提供基线。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.2

4. 文中把病变 progression 明确描述为一个 continuum：从正常 gland，到 dysplasia，再到 polyp formation，最终侵袭性 adenocarcinoma；其中 dysplasia 的标志包括 stratification、hyperchromasia、elongation 以及 gland architecture 的丢失。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.2

5. 对 adenomatous polyps 的分类也很有用：`tubular / villous / tubulovillous` 三类中，villous 和 tubulovillous 比 tubular 更容易恶变。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.3

6. 论文给出了 conventional adenocarcinoma 的 gland formation 分级口径：`well differentiated` 为 `>95%` 可识别 gland formation，`moderately differentiated` 为 `50%-90%`，`poorly differentiated` 为 `<50%`。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.3

7. 该文还明确指出 mucinous subtype 更常见于 `right-sided CRC`，通常分期更高、预后更差；这和后续一些专门研究 mucinous adenocarcinoma 的论文能互相印证。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.1, p.4

8. 在组织学特征层面，文中提到 adenocarcinoma 常见 `desmoplasia` 和 `dirty necrosis`，这些词在写病理背景和讨论恶性结构破坏时很有用。
   - 来源：`Classification_and_Histological_Characteristics_of_Colorectal_Cancer_2024.pdf`, p.4

对当前实验最有用的落地点：

- 这篇可以明显增强你引言第一段和病理背景部分的专业感。
- 也适合在 discussion 里解释“模型为什么要关注 gland architecture disruption，而不仅是像素级轮廓”。

### 2.13 `Clinicopathological_Characteristics_and_Outcomes_of_Colorectal_Mucinous_Adenocarcinoma_2024.pdf`

用途定位：

- 这篇很适合补“mucinous adenocarcinoma 是什么、为什么它在 gland morphology 上值得额外关注”。

可直接提取的关键信息：

1. 论文背景部分明确给出定义：`mucinous adenocarcinoma (MAC)` 指肿瘤中 `>=50%` 组织由 extracellular mucus 构成；若 mucinous component 在 `5%-50%`，则归为 `ACWM`。
   - 来源：`Clinicopathological_Characteristics_and_Outcomes_of_Colorectal_Mucinous_Adenocarcinoma_2024.pdf`, p.1, p.2

2. 文中总结了 MAC 与 NMAC 的典型差异：MAC 更常见于 `proximal colon`，肿瘤更大、`T stage` 更晚、转移更多、`dMMR` 更常见，且术前 `CEA` 更易升高。

   - 来源：`Clinicopathological_Characteristics_and_Outcomes_of_Colorectal_Mucinous_Adenocarcinoma_2024.pdf`, p.3

   - 来源：`Clinicopathological_Characteristics_and_Outcomes_of_Colorectal_Mucinous_Adenocarcinoma_2024.pdf`, p.1, p.5, p.6

5. 论文还强调：预后差异尤其集中在 `left-sided` 或 `stage III/IV` 的 MAC 患者中。

6. 另外一条很实用的信息是：对于 `stage II with high-risk factor` 和 `stage III` 的 MAC，adjuvant chemotherapy 仍然能带来 `OS / CSS / RFS` 改善。
   - 来源：`Clinicopathological_Characteristics_and_Outcomes_of_Colorectal_Mucinous_Adenocarcinoma_2024.pdf`, p.1

对当前实验最有用的落地点：

- 这篇能帮助你写清楚：并不是所有 CRC gland morphology 改变都一样，mucinous 亚型本身就带来不同病理与预后背景。
- 如果你后面在结果分析里发现某些粘液丰富、边界不规则样本更难分，这篇很适合作为讨论支撑。

1. 引言指出：CRC 一旦出现肝转移，预后明显恶化，`stage IV` 且伴肝转移的 `5-year survival` 仅约 `12%`，且约 `40%-50%` 患者诊断时无法手术。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.2

2. 文章基于单细胞图谱在 `NOR / CRC / CRCM / LM` 样本中识别出 `7` 类主要细胞：epithelial、T cells、B cells、macrophages、plasma cells、fibroblasts、mast cells。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.4

3. 最值得保留的机制结果是：在 CRC 肝转移过程中，`MIF` 与其受体 `CD74/CXCR4` 的细胞通讯显著增强，提示它可能驱动 invasion 和 migration。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.1, p.11, p.12

4. 上皮细胞层面，`TPM2`、`RPS17`、`TNNT1` 在转移相关样本中升高，而 `SPINK4` 下降，这些变化被作者认为与恶性进展和肝转移相关。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.1, p.8, p.10, p.12

5. 伪时序分析也支持这一点：随着疾病进展，`TPM2/RPS17/TNNT1` 持续上升，而 `SPINK4` 先升后降，说明这些基因确实伴随 metastasis trajectory 改变。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.9, p.10

6. 预后模型方面，作者用 `TCGA` 的 `515` 例作训练、`GEO` 的 `127` 例作验证，AUC 大致在 `0.626-0.698` 范围，说明模型有一定区分度但仍是中等水平。
   - 来源：`Construction_of_a_Prognostic_Model_for_CRC_Liver_Metastasis_Based_on_Single-Cell_Transcriptomics_2025.pdf`, p.3, p.8

对当前实验最有用的落地点：

- 这篇适合在“CRC 下游恶性进展和转移”部分引用，尤其是要把原发形态与远处转移联系起来时。
- 如果后面要解释 `SPINK4` 或与上皮分化相关信号的意义，这篇也很有参考价值。

### 2.16 `Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`

用途定位：

- 这篇很适合补“`beyond MSI` 的免疫治疗响应预测”。
- 它的价值在于把 CRC 免疫治疗从单一 `dMMR/MSI-H` 标签扩展到 `TMB`、共突变、TME、微生物组、`ctDNA` 等更细粒度 biomarker 框架。

可直接提取的关键信息：

1. 引言明确指出：ICI 在 CRC 中的成功主要局限于 `dMMR/MSI-H` 亚群，而 `mCRC` 中真正属于这一亚群的只占大约 `3%-5%`；但在早期病变里，新辅助 ICI 的 `pCR` 已可超过 `60%-70%`。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.2

2. 文章强调：`MSI` 虽然仍是最成熟 biomarker，但它不足以解释所有响应差异，因为并非所有 `dMMR/MSI-H` 都能从 ICI 获益，也存在 primary/secondary resistance。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.2, p.3

3. 在 `TME` 层面，作者把 `CD8+ TILs`、`TLS`、`IFN-γ signaling` 视为 favorable 信号，而 stromal barrier、metabolic barrier、Treg/MDSC/M2 macrophage 富集则是 resistance 的重要来源。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.4, p.5

4. 分子共突变方面，`BRAF V600E` 在 MSI-H 背景下并不稳定地预测更差 ICI 结局，但可能更容易出现 secondary resistance；`KRAS` 大约占 `40%` CRC，虽然在联合 ICI 里未必直接决定响应，却会通过 cytokine、MHC-I 与 `PD-L1` 调节影响免疫逃逸。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.5, p.6, p.8

5. 这篇还特别强调 `ctDNA`：早期 ctDNA 下降或清除与更好的 `PFS/OS` 相关；在新辅助 pembrolizumab 的 MSI-H 肿瘤中，ctDNA 清除者 `3-year EFS` 可达 `92%`，而持续阳性者仅约 `20%`。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.12

6. 微生物组部分也很有启发：`Fusobacterium nucleatum` 并非单向度坏标志物，它在不同代谢背景下既可能促进免疫逃逸，也可能在特定条件下提高 PD-1 阻断敏感性。
   - 来源：`Decoding_Immunotherapy_Response_in_Colorectal_Cancer_Translational_Insights_Beyond_MSI_2026.pdf`, p.11

对当前实验最有用的落地点：

- 这篇很适合当你要写“为什么不能只用 MSI 一个标签判断免疫治疗反应”时引用。
- 它也能把你前面零散提到的 TME、TLS、ctDNA、共突变整合成一个更完整框架。

### 2.17 `Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`

用途定位：

- 这篇适合补 `first-line mCRC` 系统治疗全景，尤其是 targeted therapy 和 immunotherapy 的相对排序。
- 它的价值在于提供比较清楚的“哪种组合更强、但毒性如何”的汇总视角。

可直接提取的关键信息：

1. 该网络 meta-analysis 纳入 `61` 项 RCT、共 `20,579` 例患者，对 `30` 种一线系统治疗方案做了贝叶斯比较。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.1, p.3

2. 全人群里，`FOLFOXIRI + bevacizumab + atezolizumab` 在 `PFS` 和 `OS` 上排名最高，分别给出大约 `HR 0.19` 和 `HR 0.48` 的优势。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.1, p.4, p.8, p.12

3. 但该方案并不是“无代价最优”：文章同时指出它的 `grade >=3 AEs` 排名也靠前，提示更强疗效通常伴随更高毒性。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.1, p.9, p.11, p.12, p.15

4. 在 `RAS/BRAF wild-type` 亚组中，`FOLFOXIRI + cetuximab` 在 `PFS` 和 `OS` 上排名更优，说明分子分层后最优方案会改变。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.1, p.12, p.14

5. 对 `>=70` 岁患者，`trifluridine-tipiracil + bevacizumab` 在 `PFS/OS` 上有一定优势，但严重不良反应发生率也更高，因此更依赖 geriatric assessment。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.1, p.11, p.12, p.16, p.17

6. 文章还再次强调：免疫治疗的明确获益人群仍主要是 `MSI-H/dMMR mCRC`，而大多数 `MSS/pMMR` 患者对 ICI 依旧有限敏感。
   - 来源：`Efficacy_and_Safety_of_First-Line_Targeted_and_Immunotherapy_for_Metastatic_CRC_A_Network_Meta-Analysis_2025.pdf`, p.3, p.13

对当前实验最有用的落地点：

- 这篇适合在临床治疗背景里引用，尤其用于概括 mCRC 一线方案的强度-毒性权衡。
- 如果后面要说明为什么免疫治疗组合常常依赖分子分层与年龄分层，这篇很好用。

### 2.18 `Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`

用途定位：

- 这篇很适合补 `early-onset CRC` 的病理与分子背景。
- 它的价值在于把“年轻患者 CRC 更凶险”这类泛泛表述拆成可引用的组织学和分子特征。

可直接提取的关键信息：

1. 摘要明确给出定义：`early-onset CRC` 指 `<50` 岁诊断的 CRC，而且其发病率近几十年持续上升。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.1

2. 引言进一步给出流行病学口径：美国 `SEER` 数据显示 early-onset CRC 自上世纪 90 年代中期以来约以每年 `2%-3%` 的速度增长。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.2

3. 这篇是系统综述加 meta-analysis，共筛到 `149` 篇文章，重点比较 early- vs. late-onset CRC 在突变、MSI、组织学亚型和免疫特征上的差异。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.1

4. 分子层面最核心的结果是：early-onset CRC 更少见 `KRAS`、`BRAF`、`APC` 突变，也倾向更少见 `NRAS`，但更常见 `PTEN` 与 `TP53` 突变。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.1, p.5, p.9

5. 在分子通路层面，文章指出 early-onset CRC 的 `MSI` 概率更高，合并效应量约为 `OR 1.31`，而 `CIMP-high` 反而更少见。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.6

6. 组织学方面，early-onset CRC 更常见 `high-grade`、`mucinous` 和 `signet ring` 表型，其中 signet ring 的关联最强。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.1, p.7, p.10

7. 文章也提醒：early-onset CRC 的免疫微环境结果并不完全一致，有些研究提示免疫浸润更高，也有研究提示效应 `CD8+ T cells` 更低、`Treg` 更高，因此不能把它简单当成统一免疫表型。
   - 来源：`Histological_and_Molecular_Characteristics_of_Early-Onset_Colorectal_Cancer_2024.pdf`, p.7

对当前实验最有用的落地点：

- 这篇能帮助你把年轻 CRC 亚型写得更具体，尤其适合接在组织学异质性或特殊亚型背景之后。
- 如果后面要讨论某些高分级、黏液样、印戒细胞样样本的临床含义，它很有支撑力。

### 2.19 `Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`

用途定位：

- 这篇很适合补 CRC 免疫治疗“为什么有效、又为什么常常无效”的背景。
- 它的价值在于把 `MSI-H 有效 / MSS 受限 / 获得性耐药` 这几个临床讨论常用句，变成更完整的论证链。

可直接提取的关键信息：

1. 摘要明确指出：`ICIs` 在 `MSI-H CRC` 中临床效果显著，但在 `MSS CRC` 中疗效有限，而且即便初始有效也可能出现 `acquired resistance`。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.1

2. 文章给出一个重要背景：约 `20%` CRC 在诊断时已出现转移，而 `mCRC` 的 5 年生存率仍然只有约 `14%`。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.2

3. 该文解释 `MSI-H/dMMR` 更容易从 ICI 获益的原因：更高的 neoantigen 负荷、更丰富的 `CD8+ T cells` / `Th1` 浸润，以及更强的肿瘤免疫原性。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.2, p.5

4. 临床试验表显示：在 `pMMR` / `MSS` 人群中，pembrolizumab 的 `ORR` 可以低至 `0%`，而在 `dMMR` 人群中则可达到更高响应水平。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.4

5. 获得性耐药机制部分总结得很清楚：包括 `loss of tumor antigen expression / impaired presentation`、`reduced IFN-γ response`、以及免疫抑制性 cytokine/metabolite dysregulation。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.5, p.6, p.7

6. 文章同时指出：提高 MSS CRC 免疫治疗效果的策略包括 `chemotherapy + ICI`、`dual immunotherapy`、`anti-angiogenic therapy + ICI` 等联合治疗路径。
   - 来源：`Immune_Checkpoint_Inhibitors_in_Colorectal_Cancer_Limitations_and_Challenges_2024.pdf`, p.9

对当前实验最有用的落地点：

- 这篇很适合放在临床背景和讨论部分，解释为什么 CRC 的免疫治疗并不是“普遍有效”。
- 如果后面你要讨论特殊样本免疫环境与分割难度、下游价值之间的联系，它也很好用。

### 2.20 `Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`

用途定位：

- 这篇适合补 CRC 的肿瘤微环境与免疫背景。
- 它的价值在于让你后面讨论腺体结构破坏时，不只停留在 morphology，还能自然过渡到 stromal/immune context。

可直接提取的关键信息：

1. 摘要开头就强调：CRC 结局不仅由 genetic alteration 决定，也深受 `tumor microenvironment (TME)` 影响；TME 包括 stromal cells、endothelial cells、ECM、gut microbiota 与多类 immune cells。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.1

2. 文章把 CRC 明确区分为免疫学上的 `hot` 与 `cold` 肿瘤：前者 immune infiltration 更高、细胞毒活性更强，后者则更偏 immune exclusion；这和 `MSI-H / CMS1` 与 `MSS / CMS4` 的差异密切相关。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.4

3. 文中对 macrophage 的总结很实用：`M1` infiltration 与更好 overall survival 相关，而 `M2` macrophage 与 immunosuppression、metastasis 和较差预后相关。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.5, p.6

4. `NK cells` 在 CRC 中也不是可有可无：其浸润在 metastatic 和 `CMS4` CRC 中明显更低，并与不良结局相关；而 `MSS CRC` 中的 `PD-L1`、`TGF-β` 等免疫抑制环境会削弱 NK 活性。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.6

5. 适应性免疫部分指出：`TLS`、`CD20+ B cells` 和有效的 `CD8+ T cells` 往往和更好的 disease-free survival 及 ICI response 相关，但 `Bregs`、`Tregs`、`MDSCs` 富集时则提示更强免疫逃逸。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.10, p.12

6. 文章还给出一个非常适合讨论的总判断：约 `85%` 的 CRC 属于 `MSS`，通常具有低突变负荷、低 T-cell infiltration 和免疫抑制性 TME，因此对 ICI 反应差，需要通过联合治疗把“cold”肿瘤转热。
   - 来源：`Immunological_Landscape_of_Colorectal_Cancer_TME_Cellular_Players_and_Immunotherapeutic_Opportunities_2025.pdf`, p.13, p.14

对当前实验最有用的落地点：

- 这篇能把你的 discussion 从“结构难分”进一步延伸到“这些结构变化所处的免疫环境也影响疾病行为”。
- 如果后面要写临床意义或未来下游分析，它是很好的 TME 背景支撑。

### 2.21 `Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`

用途定位：

- 这篇适合补 `ICD-related signature`、多组学风险分层和潜在用药优化。
- 它比前面的 ICD 论文更偏方法学和多组学整合，适合用来补“ICD 如何落到可操作的风险模型”。

可直接提取的关键信息：

1. 引言明确指出：`ICD` 可以通过释放 `DAMPs` 和肿瘤抗原促进 DC 成熟与细胞毒性 T 细胞浸润，从而有机会逆转免疫抑制性 TME、提高 immunotherapy sensitivity。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.2

2. 文章整合了单细胞和 bulk RNA 数据：在 `41,958` 个单细胞里，作者发现 macrophages 的 ICD activity 最高，其次是 T cells。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.7

3. 方法上很有代表性：作者比较了 `101` 种机器学习组合，最终用 `StepCox[forward] + RSF` 构建出 `11-gene ICDRS`，并显示其在多个队列中比 `60` 个既有 CRC signature 表现更好。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.9, p.10, p.21

4. 该模型对 OS 的预测性能较强，部分时间点 `AUC` 达到 `0.84-0.88`，而结合临床因素后的 nomogram 在 `1/2/3-year` 预测里 `AUC` 甚至超过 `0.9`。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.8, p.9, p.12

5. 免疫响应分析显示：`low-risk` 组有更高 `IPS`、更低 `TIDE`，更像 immunotherapy responder；而 `high-risk` 组虽然 checkpoint expression、stromal/immune score 以及 `TMB` 更高，却更容易表现为 immune evasion。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.10, p.11, p.12, p.13, p.14

6. 文章还识别出 `SPP1+/SLC11A1+ macrophages` 这一潜在关键亚群，并把 `Olaparib` 提为可能对高风险 CRC 更有利的候选药物。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.18, p.19, p.20

7. 功能实验部分则提示 `PLA1A` 在 CRC 中上调，且可能促进 proliferation 和 invasion，说明这个 ICDRS 不只是统计模型，也有一定生物学可解释性。
   - 来源：`Integrated_Multi-Omics_and_Machine_Learning_Reveal_an_ICD-Related_Signature_for_CRC_2025.pdf`, p.16, p.17

对当前实验最有用的落地点：

- 这篇很适合在 ICD、TME 和个体化治疗之间做桥接。
- 如果后面要写“高免疫分数不等于高响应”这类更细的免疫生物学讨论，它很有参考意义。

### 2.22 `Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`

用途定位：

- 这篇适合补 `Immunoscore` 相关的更细化讨论。
- 它的核心价值在于提醒你：单纯照搬 Immunoscore 的 percentile averaging 可能并不是最生物学合理的做法。

可直接提取的关键信息：

1. 摘要说明：作者研究 `intra-tumoral CD3+/CD8+` 计数与改良评分，目的是改进传统 `Immunoscore™` 的预后价值并更好对齐底层癌症生物学。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.1

2. 文中直接指出 Immunoscore 的几个局限：成本更高、受 `intra-tumoral heterogeneity` 影响、且忽略了其他潜在重要的免疫变量。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.1, p.2

3. 研究纳入 `201` 例切除 CRC，通过定量 `CD3/CD8` IHC，计算类似 Immunoscore 的 `I-score`，并探索 cell density、difference、ratio、covariance 等多种替代指标。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.1, p.3

4. 作者给出了有用的 cutoffs：`CD3+ >254/mm²` 与 `CD8+ >128/mm²` 都和更好 DSS 相关，而更高的 `covariance` 则提示更差预后。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.7

5. 一个很关键的结论是：`I-score` 虽然与预后有关，但在控制 `pT/pN/pM` 分期后并非独立预测因子；相反，`CD3+:CD8+ percentile ratio` 才是 stage-independent predictor。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.8, p.9

6. 结论部分进一步强调：`CD8+ density alone` 已经和组合指标一样有信息量，而相对密度关系可能比简单平均 percentile 更有生物学意义。
   - 来源：`Intra-Tumoral_Lymphocyte_Scoring_in_Colorectal_Cancer_2024.pdf`, p.10

对当前实验最有用的落地点：

- 如果你后面要写 immune contexture 或预后分析，这篇能帮助你避免机械复述 Immunoscore。
- 它也说明了肿瘤内部空间异质性本身就是一个应当被尊重的病理事实。

### 2.23 `Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`

用途定位：

- 这篇适合补 `TLS` 在 CRC 免疫治疗预测中的作用。
- 它的亮点是把 `TLS quantity + maturity + intratumor location` 结合起来做了一个可用的病人评分。

可直接提取的关键信息：

1. 摘要和引言都指出：虽然 `PD-1` 抗体已经改变了 CRC 治疗，但即使在 `dMMR` CRC 中，实际也不到一半患者会稳定获益，因此需要更准确的免疫预测指标。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.1, p.2

2. 论文用多重免疫荧光评估 `TLS` 的数量和成熟度，并按 `Grade 1/2/3` 区分：从单纯淋巴聚集到具有 `CD21` 网络，再到出现 `Bcl-6+` 生发中心反应。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.3

3. 研究显示：无论 whole tumor、peritumor 还是 intratumor，`dMMR` 患者的 TLS 数量都高于 `pMMR`；而 `TLS maturity` 的显著差异主要出现在 `intratumor` 层面。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.4, p.5

4. 作者基于 `TLS number` 和 `TLS maturity` 建立了一个简单 patient score（0/1/2）；其中 `intratumor TLS score >1` 的比例与文献报道的临床响应率接近。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.5

5. 在接受 `anti-PD1` 治疗的患者中，`intratumor TLS score >1` 与更好的 response 显著相关；在 `pMMR` 人群中，这个评分对反应预测尤其明显。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.6, p.7

6. 生存分析显示：在 `pMMR` 患者中，`score >1` 对更长的 `irPFS` 和 `OS` 有显著预测价值，且 ROC 分析得到 `AUC = 0.815`。
   - 来源：`Intratumor_Tertiary_Lymphatic_Structure_Predicts_Prognosis_and_Immunotherapy_Response_in_CRC_2024.pdf`, p.6, p.8

对当前实验最有用的落地点：

- 这篇很适合支持你把 CRC 结构病理与免疫微环境更紧密地连起来。
- 如果后面要讨论哪些空间组织结构可能预示免疫治疗反应，这篇非常有价值。

### 2.24 `Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`

用途定位：

- 这篇很适合补 `non-metastatic dMMR/MSI-H CRC` 新辅助免疫治疗的总体现状。
- 它的价值在于给出比较完整的 `pCR / MPR / ORR / toxicity` 汇总口径，适合写临床背景。

可直接提取的关键信息：

1. 摘要说明：该系统综述纳入 `21` 篇研究，共 `628` 例 `non-metastatic dMMR CRC`。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.1

2. 文中同时回顾了一个关键背景：`dMMR/MSI-H` 约占 `15%` 的非转移性 CRC，而这类肿瘤对传统新辅助化疗往往反应差，因此免疫治疗特别值得关注。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.2

3. 最重要的汇总结果是：新辅助免疫治疗后的 `pCR` 为 `320/480 = 66.6%`，`MPR` 为 `388/452 = 85.8%`，`ORR` 为 `89.7%`。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.1, p.6, p.8

4. 分层分析显示：`T1-T3` 与 `T4a-T4b` 的 `pCR` 率相近；对于局部进展的 `T4` 患者，文章仍报告了相当高的 `pCR`，提示晚一些的局部病灶也可能明显获益。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.1, p.8, p.9

5. 安全性方面，文中汇总 `irAEs` 约为 `59%`，术后并发症约为 `19.3%`，且多数 `irAEs` 为 `grade I-II`，说明总体可控。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.6, p.9

6. 一个很实用的结论是：`immune monotherapy` 的 `pCR` 与联合治疗相近，但毒性更低，因此作者倾向把单药作为更高效益-毒性比的优先方案。
   - 来源：`Neoadjuvant_Immunotherapy_for_Non-Metastatic_dMMR_Colorectal_Cancer_A_Systematic_Review_2025.pdf`, p.10, p.11, p.12, p.13

对当前实验最有用的落地点：

- 这篇适合放在 dMMR/MSI-H CRC 的临床意义讨论里，说明为什么免疫相关结构背景值得关注。
- 它也能支撑“局部非转移性 CRC 中，病理反应已经可以非常深”的论述。

### 2.25 `Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`

用途定位：

- 这篇很适合补 `pMMR` 结肠癌新辅助免疫治疗的少数阳性证据。
- 它的重要性在于说明：虽然 `pMMR/MSS` 通常被认为对 ICI 不敏感，但在早期局部阶段并不是完全无响应。

可直接提取的关键信息：

1. 摘要和引言都强调：转移性 `pMMR CRC` 对 `ICB` 疗效有限，但 `pMMR` 占非转移性结肠癌的大多数，因此探索其新辅助免疫治疗很有价值。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.2, p.3

2. 该研究来自 `NICHE` 平台试验的 `pMMR cohort`，共 `33` 例接受治疗，`31` 例纳入疗效分析。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.3, p.4

3. 结果显示：总响应率为 `8/31 = 26%`；在接受手术的 `30` 例中，`MPR` 为 `6/30 = 20%`，`pCR` 为 `3/30 = 10%`。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.4

4. 响应并不只出现在最低风险病例中：`8` 个 responder 里有一半是 `stage III`，说明临床高风险 `pMMR` 肿瘤也可能对短程新辅助 ICB 产生深病理反应。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.4

5. 文章还发现：加入 `celecoxib` 并没有改善 response rate，也未观察到明确的免疫调节增益，因此单纯加 `COX-2` 抑制剂并不是明显有效的增强路径。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.4, p.7, p.8

6. 生物学分析提示 responder 更常见 `TP53 mutation`、更高的 genomic instability 与 whole-genome duplication，这可能对应一个更容易产生免疫反应的亚群。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.5, p.9, p.10

7. 一个很关键的负结果是：传统上常被期待的 `IFN-γ` 和 `baseline CD8+ infiltration` 并不能稳定区分 responder 与 non-responder，说明 pMMR 场景下机制并不简单。
   - 来源：`Neoadjuvant_Immunotherapy_in_Mismatch-Repair-Proficient_Colon_Cancers_2025.pdf`, p.6, p.9

对当前实验最有用的落地点：

- 这篇很适合用来避免把 `pMMR/MSS = 完全不响应免疫治疗` 写得过于绝对。
- 如果你后面想把形态、基因不稳定性和免疫反应联系起来，这篇是很有价值的桥接文献。

### 2.26 `Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`

用途定位：

- 这篇适合补 `MSI-related genes + clinical features` 联合预后建模的背景。
- 它对你最有用的是说明：单独讲 MSI 或单独讲临床分期都不够，整合后预后分层明显更好。

可直接提取的关键信息：

1. 摘要说明：作者整合 `TCGA` 与 `GEO` 的 `483` 例 CRC 数据，尝试把 `MSI-related gene modules` 和临床信息联合起来做精细化预后评估。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.1

2. 文章通过差异分析和 `WGCNA` 先筛出 MSI 相关模块，再建立三个模型：`Model I` 为基因模型，`Model II` 为临床模型，`Model III` 为整合模型。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.1, p.3, p.4

3. 性能上，`Model I` 的 `AUC = 0.724`，`Model II` 的 `AUC = 0.684`，单独看都还可以，但都不如整合模型。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.1

4. `Model III` 把 `6` 个 MSI 相关基因与 `stage`、`M` 等临床参数整合后，`AUC` 提升到 `0.825`，而且高低风险组的 Kaplan-Meier 曲线差异显著。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.1, p.5

5. 外部验证集里，`Model III` 仍然保持 `AUC = 0.767`，说明这个 MSI+clinical 的整合框架有一定稳健性。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.1, p.5

6. 讨论部分也强调：MSI 不只是一个孤立标签，它更像连接下游分子事件与最终生存结局的中间里程碑，因此适合纳入综合模型而不是单独讨论。
   - 来源：`Precision_Prognosis_of_Colorectal_Cancer_A_Multi-Tiered_Model_Integrating_MSI_Genes_and_Clinical_Parameters_2024.pdf`, p.6, p.7, p.8

对当前实验最有用的落地点：

- 这篇适合在预后分层或临床意义部分引用，说明单一生物标志物往往不足。
- 如果后面要把病理结构、MSI 和临床结局放在一个框架里讨论，它能提供比较好的方法学参照。

### 2.27 `Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`

用途定位：

- 这篇适合补 `ICD-related genes`、免疫治疗反应和 `ferroptosis` 之间的桥接背景。
- 它的价值不在直接指导腺体分割，而在于帮你把 CRC 的免疫响应、药物敏感性和细胞死亡机制串起来。

可直接提取的关键信息：

1. 摘要说明：作者基于 `TCGA/GEO` 的大样本表达数据，系统分析了 `immunogenic cell death (ICD)` 相关基因在 CRC 中的表达、预后和治疗意义，并特别讨论了 `ferroptosis inducer` 的潜在价值。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.1, p.2

2. 文中首先强调一个概念背景：`ICD` 通过 `DAMPs` 释放可诱导 `tumor-specific CD8+ T cells` 和免疫记忆，因此它天然和免疫治疗响应问题相关。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.2, p.3

3. 该研究对 CRC 患者做了 `consensus clustering`，把人群分成不同 ICD 相关分子亚型，并显示不同亚型对应不同生存结局和临床特征。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.4, p.15

4. 文章进一步结合 `CIBERSORT`、`ssGSEA` 和单细胞数据，显示不同 ICD 亚型在 `plasma cells`、`CD8+ T cells`、macrophage 等免疫浸润上存在明显差异，说明 ICD 信号与免疫微环境异质性密切相关。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.7, p.9, p.15

5. 预后建模部分构建了一个 `16-gene` 风险模型；高风险组在训练集和外部验证集里都表现出更差生存，且 `3/5/10-year AUC` 处于可用水平。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.8, p.9

6. 药敏分析提示：预后较差的亚群对多种药物表现出更高 `IC50`，而文中讨论认为 `ferroptosis inducers` 可能通过增强 ICD 来改善冷肿瘤的免疫反应。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.10, p.13, p.14, p.16, p.17

7. 作者最后明确提出：对于对常规治疗不敏感的 CRC，`ferroptosis inducer + immunotherapy` 可能成为更精准的联合路径，但仍需要更大规模机制和前瞻性验证。
   - 来源：`Prognosis_of_CRC_and_ICD-Associated_Genes_in_Response_to_Immunotherapy_2024.pdf`, p.16, p.17

对当前实验最有用的落地点：

- 这篇适合放在免疫治疗与新机制讨论里，尤其是当你想把细胞死亡机制和免疫响应联系起来时。
- 如果后面要补“冷肿瘤如何被重新激活”的背景，它很有用。
